# Changelog

## 1.0.0

First public release of HardSub Reader.

HardSub Reader is an NVDA add-on that reads hardcoded (burned-in) subtitles from videos and announces them through speech, making subtitled content more accessible to blind and visually impaired users.

Highlights

• Uses Microsoft's OneOCR engine on supported versions of Windows for fast and accurate subtitle recognition.

• Automatically falls back to the legacy Windows OCR engine when OneOCR is unavailable, ensuring compatibility across supported Windows versions.

• Includes configurable settings for subtitle detection and recognition behavior through the NVDA Settings dialog.

• Supports customizable input gestures using NVDA's Input Gestures dialog.

• Designed to minimize repeated announcements while providing timely updates as subtitles change.
